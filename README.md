Learning Management System (LMS)

A full-stack Learning Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

Features

- User Authentication (Login/Register)
- Student Enrollment
- Course Management
- Admin Dashboard
- Razorpay Subscription Payments
- Protected Routes
- Profile Management
- Responsive UI

Tech Stack

Frontend

- React.js
- Redux Toolkit
- React Router
- Tailwind CSS

Backend

- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Razorpay API

Installation

Clone Repository

git clone https://github.com/Rgpraveen11/learning-management-system.git

Install Dependencies

Client:
npm install

Server:
npm install

Run Project

Client:
npm run dev

Server:
npm run dev
